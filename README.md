Sebelum ngerun program metode Secant, pastikan sudah menginstal **NumPy** terlebih dahulu.  
Gunakan perintah berikut di terminal atau command prompt:

Untuk Mac:
```bash
pip install numpy
```
Untuk Windows:
```bash
pip install numpy
```
Untuk Linux:
```bash
pip install numpy
```
Jika pip tidak dikenali:
```bash
python3 -m pip install numpy
```
